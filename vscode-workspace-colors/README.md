# Workspace Colors

Easily distinguish between VS Code windows by applying custom color themes to the title bar, activity bar, and status bar.

## Features

- Visual color picker with categorized color palettes
- Live preview on hover - see colors applied instantly
- Automatic bright/dark color pairing for title bar states
- Persists per-workspace - each project can have its own color

## Usage

1. Open the Command Palette (`Cmd+Shift+P` / `Ctrl+Shift+P`)
2. Run **Pick Workspace Color Theme**
3. Hover over colors to preview, click to select
4. Click **Save** to apply

To remove colors, run **Clear Workspace Color Theme** or click **Restore Default** in the picker.

## Commands

| Command | Description |
|---------|-------------|
| `Pick Workspace Color Theme` | Open the color picker |
| `Clear Workspace Color Theme` | Remove workspace colors |
